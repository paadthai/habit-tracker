# 🐇 Habit Tracker — 12간지 윈도우95 스타일 웹앱 기획서

## 개요

개인 습관 추적 PWA(Progressive Web App). 앱스토어 등록 없이 폰 홈화면에 설치해서 사용. 모든 데이터는 브라우저 localStorage에 저장하며 외부 서버/로그인 불필요.

---

## 기술 스택 (권장)

- **HTML + CSS + Vanilla JS** 단일 파일 또는 소규모 멀티파일
- **저장소:** `localStorage`
- **호스팅:** GitHub Pages (무료, URL 생성됨)
- **PWA:** `manifest.json` + Service Worker로 오프라인 지원 및 홈화면 설치 가능

---

## 디자인 컨셉

### 윈도우95 UI 스타일
- 회색(`#c0c0c0`) 배경, 베벨 테두리 (`inset` / `outset` box-shadow)
- 픽셀 폰트 (예: `Press Start 2P`, MS Sans Serif 계열)
- 클릭 시 버튼이 눌리는 애니메이션
- 타이틀바 (파란색 그라디언트), 최소화/최대화/닫기 버튼 스타일
- 창(window) 형태의 UI 컴포넌트

### 픽셀아트 12간지 동물
- 각 동물을 **픽셀아트 SVG 또는 Canvas**로 구현
- 달성률에 따라 4단계로 채색:
  - `0~25%` → 윤곽선만 (흑백 점선)
  - `26~50%` → 25% 색칠
  - `51~75%` → 75% 색칠
  - `76~100%` → 풀컬러 + 완성 효과 (반짝임 등)

## 화면 구성

### 1. 메인 화면 — 주간 습관 테이블

| 열 | 내용 |
|----|------|
| 1열 | 습관 이름 (텍스트 입력) |
| 2열 | 목표 횟수 (숫자 입력) |
| 3열 | 기준 (`이상` / `이하` 드롭다운) |
| 4~10열 | 월~일 체크박스 (각 칸은 네모 박스 클릭으로 ✅ 토글) |

- 습관은 **언제든 추가/삭제 가능**
- 최대 10개 이내 권장 (UI 고려)
- 각 행 우측에 삭제 버튼 (`[X]`)
- 하단에 `[+ 습관 추가]` 버튼

### 2. 대시보드 (상단 고정 또는 탭)

- **이번 주 전체 달성률** (% 숫자 + 프로그레스 바)
- **습관별 달성률** 리스트 (바 형태)
- **이번 달 동물** 픽셀아트 — 달성률에 따라 실시간 채색 상태 반영

### 3. 연간 갤러리 화면

- 윈도우95 "내 컴퓨터" 스타일 아이콘 배열
- 12개 칸 = 12개월
- 각 칸: 동물 픽셀아트 + 월 이름 + 달성률 %
- 완료된 달 → 풀컬러 / 미래 달 → 잠금 또는 흑백
- 클릭 시 해당 월 상세 통계 팝업 (윈도우95 모달 스타일)

## 핵심 로직

### 달성률 계산
```
주간 달성률 = (체크된 칸 수 / 전체 칸 수) × 100
월간 달성률 = 해당 월 전체 주의 달성률 평균
```

### 동물 채색 단계
```
0~25%   → stage 1 (윤곽선)
26~50%  → stage 2
51~75%  → stage 3
76~100% → stage 4 (풀컬러)
```

### 월 마감 처리
- 매달 1일에 전월 데이터를 `monthlyStats`에 자동 저장
- 갤러리에서 확인 가능

## PWA 설치 방법 (완성 후)

1. GitHub Pages에 업로드
2. 폰에서 URL 접속
3. **iOS:** 사파리 → 공유 버튼 → 홈 화면에 추가
4. **Android:** 크롬 → 메뉴 → 홈 화면에 추가
5. 이후 앱 아이콘으로 실행 가능
